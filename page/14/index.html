枚举问题：特殊密码锁 2018-09-07|MOOC笔记C++

总时间限制:
 1000ms

内存限制:
 1024kB

**描述
**有一种特殊的二进制密码锁，由n个相连的按钮组成（n<30），按钮有凹/凸两种状态，用手按按钮会改变其状态。
然而让人头疼的是，当你按一个按钮时，跟它相邻的两个按钮状态也会反转。当然，如果你按的是最左或者最右边的按钮，该按钮只会影响到跟它相邻的一个按钮。
当前密码锁状态已知，需要解决的问题是，你至少需要按多少次按钮，才能将密码锁转变为所期望的目标状态。

**输入
**两行，给出两个由0、1组成的等长字符串，表示当前/目标密码锁状态，其中0代表凹，1代表凸。

**输出
**至少需要进行的按按钮操作次数，如果无法实现转变，则输出impossible。

**样例输入
**011
000

**样例输出
**1

来自 <http://cxsjsxmooc.openjudge.cn/2018t2winterw1/001/>

解题思路：
0和1组成的字符串，可以选择用一维数组表示密码锁。
对于密码锁状态变换的操作，需要判断当前密码是0还是1，进行取反操作。
对变换密码的过程，需要明确第一个按钮是否按下。当第一个按钮的操作情况确定后，之后所有按钮的操作都是确定的。即，当第一个按钮操作完成以后，从第二个开始扫描：若发现前一个按钮没有达到目的状态，必须通过当前按钮去改变前一个按钮的状态，直到扫描完全部按钮为止。对于是否能够完成任务的判定，只需要到最后一个按钮时判断最后一个按钮的状态是否符合条件即可。若可以完成任务，但不一定是最优解，因为第一个按钮的操作方案有两种，需要对第一个按钮的两种方案都做枚举。

例题：完美立方

形如a^3=b^3+c^3+d^3的灯饰被称为完美立方等式，例如

12^3=6^3+8^3+10^3

编写一个程序对任给的正整数N<=100寻找所有的四元组（a,b,c,d）满足公式，其中a,b,c,d大于1小于等于N，且b<=c<=d

输入：一个正整数N

输出：每行一个完美立方

Cube=a,Triple=(b,c,d)

解题思路

四重循环abcd，a外层，d里层，每一层从小到大枚举。

a范围【2，N】，b范围【2，a-1】，c范围【b，a-1】，d范围【c，a-1】

LeetCode题解707.设计链表 2018-09-04|C++

LeetCode题解707.设计链表

设计链表的实现。可以选择使用单链表或双链表。单链表中的节点应该具有两个属性：val 和 next。val 是当前节点的值，next 是指向下一个节点的指针/引用。如果要使用双向链表，则还需要一个属性 prev 以指示链表中的上一个节点。假设链表中的所有节点都是 0-index 的。

在链表类中实现这些功能：

  • get(index)：获取链表中第 index 个节点的值。如果索引无效，则返回-1。
  • addAtHead(val)：在链表的第一个元素之前添加一个值为 val 的节点。插入后，新节点将成为链表的第一个节点。
  • addAtTail(val)：将值为 val 的节点追加到链表的最后一个元素。
  • addAtIndex(index,val)：在链表中的第 index 个节点之前添加值为 val  的节点。如果 index 等于链表的长度，则该节点将附加到链表的末尾。如果 index 大于链表长度，则不会插入节点。
  • deleteAtIndex(index)：如果索引 index 有效，则删除链表中的第 index 个节点。

分析：

基本上就是实现单链表的各种常规操作。一些知识点和需要注意的地方在之后链表中再进行解释。

解决方案源码（非完整项目代码）：

class MyLinkedList {
public:
/** Initialize your data structure here. */
MyLinkedList() {
LinkedList = 0;
}
ListNode *LinkedList;
/** Get the value of the index-th node in the linked list. If the index is invalid, return -1. */
int get(int index) {
int i = 0;
ListNode *head = LinkedList;
while (head&&i < index) {
head = head->next;
i++;
}
if (head&&i == index) return head->val;
else return -1;

}

/** Add a node of value val before the first element of the linked list. After the insertion, the new node will be the first node of the linked list. */
void addAtHead(int val) {
ListNode *head = (ListNode *)malloc(sizeof(ListNode));
head->next = LinkedList;
head->val = val;
LinkedList = head;
}

/** Append a node of value val to the last element of the linked list. */
void addAtTail(int val) {
ListNode *head = LinkedList;
ListNode *tmp = (ListNode *)malloc(sizeof(ListNode));
tmp->next = 0;
tmp->val = val;
if (!head) {
LinkedList = tmp;
return;
}
while (head->next) {
head = head->next;
}
head->next = tmp;
}

/** Add a node of value val before the index-th node in the linked list. If index equals to the length of linked list, the node will be appended to the end of linked list. If index is greater than the length, the node will not be inserted. */
void addAtIndex(int index, int val) {
int i = 0;
ListNode *head = LinkedList;
if (!head&&index == 0) {
ListNode *tmp = (ListNode *)malloc(sizeof(ListNode));
tmp->val = val;
tmp->next = 0;
LinkedList = tmp;
return;
}
while (head&&i < index - 1) {
head = head->next;
i++;
}
if (head&&head->next == 0) {
ListNode *tmp = (ListNode *)malloc(sizeof(ListNode));
tmp->val = val;
tmp->next = 0;
head->next = tmp;
}
else if (i == index - 1 && head&&head->next) {
ListNode *tmp = (ListNode *)malloc(sizeof(ListNode));
tmp->val = val;
tmp->next = head->next;
head->next = tmp;
}
}

/** Delete the index-th node in the linked list, if the index is valid. */
void deleteAtIndex(int index) {
ListNode *head = LinkedList;
int i = 0;
while (head&&i < index - 1) {
head = head->next;
i++;
}
if (head == 0) return;
if (head->next == 0 && index == 0) {//只有一个节点的情况
LinkedList = 0;
return;
}
if (head->next) {
ListNode *tmp = head->next;
head->next = tmp->next;
free(tmp);
}
}
};

Conciseness and Emphasis What is conciseness? “Concise writing mankes every woed count”(Aeron 129). NO useless words What is emphasis? “Emphatic writing leads your readers to see your main ideas both within and among sentences”(Aeron 99). NOT leave your reader guessing and wondering about the point How to achieve conciseness and emphasis? 1.Focus on the subject and verb Sentence A: The reason why most of the country shift to daylight savings time is that winter days are much shorter than summer days. Sentence B: Most of the country shift to daylight savings time because winter days are much shorter than summer days.(concise) 2.Avoid using nouns derived from verbs Sentence A: The intention of the company was to expand its workforce. A proposal was also made to diversify the backgrounds and abilities of their employees. PS: Obscures the key action of the sentence Sentence B: The company intended to expand its workforce. It also proposed to diversity the backgrounds and abilities of their employees. 3.Avoid using weak verbs “be” “make” “have” Sentence A: Ashard as this is to believe, China is now the leader of globalization. Sentence B: As hard as this is to believe, China is now leading globalization. 4.Avoid using passive voice Sentence A: It is believed by the candidate that a ceiling must be palced on the budget by Congress. Sentence B: The candidate believes that Congress must place a ceiling on the budget. 5.Avoid using empty words Examples: All things considered In a manner of speaking For all intents and purposes For the most part In my opinion Last but not least More or less As far as I am concerned As we all know that It cannot be denied that 6.Avoid using wordy expressions At all times Always At the persent time Now,yet Because of the fact that Because By virtue of the fact that Because Due to the fact that Because For the purpose of Because In the event that If In order to To In the final analysis finally 7.Begin with old information and end with new Not emphatic: a.
