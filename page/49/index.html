题目：举例分析《诗经》六义之首“风”的含义

钱钟书先生在《管锥编》中写：

言其作用,“风”者,风谏也、风教也。 言其本源,“风”者,土风也、风谣也,今语所谓地方民歌也。 言其体制,“风”者,风咏也、风诵也,系乎喉舌唇吻,今语所谓口头歌唱文学也。

即，第一层为，作用 “风”是诗经六义风雅颂赋比兴之一。尤其是在被儒家作为五经之一，其教育作用更甚。

《关雎》，后妃之德也，“风”之始也，所以风天下而正夫妇也，故用之乡人焉，用之邦国焉。“风”，风也，教也。 “是以关雎乐得淑女配君子，忧在进贤不淫色；哀窈窕，思贤才，而无伤善之心焉。是关雎之义也。——《毛诗序》

在《关雎》的教化作用： 《毛诗传》云：“窈窕，幽闲也。淑，善也。逑，匹也。言后妃有关雎之德，是幽闲贞专之善女，宜为君子之好匹。”“窈窕淑女”就是“幽”，“闲”，“善”，空幽，闲静，至善。 第二层为，本义，即各地的风土人情 如，《国风·郑风·子衿》

青青子衿，悠悠我心。纵我不往，子宁不嗣音？ 青青子佩，悠悠我思。纵我不往，子宁不来？ 挑兮达兮，在城阙兮。一日不见，如三月兮。

第三层，即为男女之间的爱情

吾闻之,凡诗之所谓风者,多出于里巷歌谣之作。所谓男女相与咏歌,各言其情者也。惟《周南》《召南》,亲被文王之化以成德,而人皆有以得其性情之正,故其发于言者,乐而不过于淫,哀而不及于伤,是以二篇独为风诗之正经。 ——[宋]朱熹《诗集传序》 天下女子有情,宁有如杜丽娘者乎!梦其人即病,病即弥连,至手画形容传于世而后死。死三年矣,复能溟莫中求得其所梦者而生。如丽娘者,乃可谓之有情人耳。情不知所起,一往而深。生者可以死,死可以生。生而不可与死,死而不可复生者,皆非情之至也。梦中之情,何必非真,天下岂少梦中之人耶?必因荐枕而成亲,待挂冠而为密者,皆形骸之论也。 ——汤显祖《牡丹亭》题辞

第二讲    情景合一

2.1 触景生情与移情于景

在中国文化中，天人合一是一个非常重要的理念，强调的是人与自然的关系。 相对于中国诗歌，就是情景合一。 景，是自然之景。 情，是人之情。 中国诗歌非常重视的就是情景合一，情为主观，景为客观，是靠情来调动。 在优秀的诗歌中，纯粹的写景是不存在的，或多或少都有作者的主管情感。   触景生情 在生活中经常发生的事例，比如，多年以前到过一个地方，现在故地重游，看到了熟悉的景物，马上就会引起某一种情感，诗如：

闺怨 王昌龄 闺中少妇不知愁， 春日凝妆上翠楼。 忽见陌头杨柳色， 悔教夫婿觅封侯。

  移情于景 把主体的情感投射到客观外务身上去，使它具有主体的情感特征，一般来说和拟人的手法相同步，诗如：

春望 杜甫 国破山河在，城春草木深。 感时花溅泪，恨别鸟惊心。 烽火连三月，家书抵万金。 白头搔更短，浑欲不胜簪。

wiggle(freq,amp,octaves=1,amp-mult=.5,t=time)

AE笔记：混合模式 2018-02-05|uncategorized|AE-AE知识点

AE混合模式 一、混合模式作用 使上下两层通过一定的交互计算，产生特殊效果。 二、混合模式操作 1、混合模式控制着图层与其下面的图层混合交互，除Alpha模式组以外，其他的模式运算都是通过修改源图层的颜色值产生效果。 2、 混合模式无法添加关键帧，也就是说图层不能在时间轴上用关键帧切换混合模式，如果有需要，可以在需要切换混合模式的时间点拆分图层，或者使用菜单栏-效果-通道-复合运算，效果和混合模式一样，可以实现混合模式随时间切换的功能。 三、混合模式使用 1、在图层上选择混合模式。 2、选择住图层后，按shift+“+”或“-”可以进行上下切换 四、混合模式介绍 1、基础模式组 Normal正常：当不透明度设置为100%，此合成模式将根据Alpha通道正常显示当前层，并且此层的显示不受其他层的影响。当不透明设置小于100%时，当前层的每一个象素点的衍射将受到其他层的影响，根据当前的不透明度值和其他层的色彩来确定显示的颜色。 Dissolve溶解：该合成模式将控制层与层之间的融合显示，因此该模式对于有羽化边界的层起到较大的影响。如果，当前层没有遮照羽化边界，或该层设定为完全不透明，则该模式几乎是不齐作用的。所以，该模式的最终效果将受到当前层Alpha通道的羽化程序和不透明度的影响。 Dancing Dissolve动态溶解模式：该模式同Dissolve相同。但它对融合区域进行了随机动画。 2、减少模式组 该模式组中性色为白色，按算法去掉白色后与下面的图层进行混合。 Darken变暗模式：以层颜色为准，比层颜色亮的像素被替换，而比层颜色暗的像素不改变。 Multiply正片叠底模式：将底色与层颜色相乘，形成一种光线透过两张叠在一起的幻灯片效果，结果呈现一种较暗的效果。任何颜色与黑色相乘得到黑色，与白色相乘则保持不变。 Linear Burn线性加深模式：类似于正片叠底，通过降低亮度，让底色变暗以反映混合色彩。和白色混合没有效果。 Color Burn颜色加深模式：使层的亮度减低，色彩加深。 Classic Color Burn经典颜色加深模式：为兼容早版本的Color Burn模式 3、添加模式组 中性色为黑色，按算法将黑色去掉后与下面的图层进行混合。 Add增加模式：将底色与层颜色相加，得到更为明亮的颜色。层颜色为纯黑色或底色为纯白色时，均不发生变化。 Lighten变亮模式： ​以层颜色为准，比层颜色暗的像素被替换，而比层颜色亮的像素不改变。 Screen屏幕模式： 将层的颜色互补色与底色相乘，呈现出一种较亮的效果。该模式与Multiply相反。 Linear Dodge线性减淡模式： 通过增加亮度来使得底层颜色变亮，以此获得混合色彩。与黑色混合没有任何效果。 Color Dodge颜色减淡：与Color Burn刚好相反，通过降低对比度，加亮底层颜色来反映混合色彩。与黑色混合没有任何效果。 Classic Color Dodge经典颜色减淡：兼容早版本的Color Dodge模式。 4、复杂模式组 Overlay叠加模式：该模式将根据底层的颜色，将当前层的像素进行相乘或覆盖。使用该模式可以导致当前层变亮或变暗。该模式对于中间色较为明显，对于高亮区域或者较暗区域影响不大。 Soft Light柔光模式：该模式创造一种柔和光线照射的效果，使亮度区域变得更亮，暗调区域变得更暗。如果层颜色比50%灰色暗，则图像会变暗。柔光的效果取决于层颜色。用纯黑色或者纯白色作为层颜色时，会产生明显较暗或者较亮的区域，但不会产生纯黑色或纯白色。 Hard Light强光模式：该模式创造一种强光照射的效果，使亮度区域变得更亮，暗调区域变得更暗。工作原理与Soft Light相同。 Liner Light线性加光模式：如果层颜色亮度高于50%灰色，则用增加亮度的方法来使得画面变亮，反之用降低亮度的方法来使画面变暗。 Vivid Light清晰加光模式：根据地色的加深或者减淡来增加或者减退层颜色对比度。如果底色比50%灰色亮，则层颜色降低，对比度变亮；如果底色比50%灰色暗，则层色提高，对比度变暗。 Din Light阻光模式：根据底色来替换层颜色。如果底色亮于50%灰色，则层颜色中更暗的像素被替换，亮于底色的像素不变。如果底色暗于50%灰色，则层颜色中更亮的像素被替换，暗于底色的像素不变。 Hard Mix强光混合模式：层颜色会和底色进行混合，通常的结果是亮色更加亮，暗色更加暗，降低填充不透明度建立多色调分色或者阈值，降低填充不透明度能使混合结果变得柔和。混合的颜色由底层颜色与混合图层亮度决定。 Difference差别模式：从底色中减去层颜色，或从层颜色中减去底色。这取决于哪个颜色的亮度较大（亮色减暗色）。与白色混合会使底色值反相，与黑色混合不产生变化。 Classic Difference经典差值：兼容老版本的差别模式。 Exclusion排除模式：创建一种与Difference类似但对比度较低的效果。 Hue色相模式：用底色的光度、饱和度和层颜色的色相创建结果颜色。 Saturation饱和度模式：用底色的亮度、色相和层颜色的饱和度创建结果颜色。如果底色为灰度区域，不会引起任何变化。 Color颜色模式：用底色的亮度和层颜色的饱和度、色相创建结果颜色。可以保护图像中的灰色色阶。 Luminosity高度模式：用底色的色相、饱和度和层颜色亮度的创建结果颜色。效果与颜色模式相反。该模式是除了Normal外唯一能完全消除纹理背景干扰的模式。 5、Alpha通道模式 上面所列的层模式，是通上下层的颜色通道混合产生影响变化，而下层的层模式则是通过层的Alpha通道影响混合变化。 Stencil Alpha    Alpha通道模式：该模式可以穿过Stencil层的Alpha通道显示多个层。 Stencil Luma亮度模式：该模式可以穿过Stencil层的像素显示多个层。当使用此模式时，层中较暗的像素比较亮的像素更透明。 Slihouette Alpha   Alpha通道轮廓：该模式可以通过层的Alpha通道在几层间切出一个洞。 Slihouette Luma亮度轮廓：该模式可以通过层上的像素的亮度在几层间切出一个洞，使用此模式，层中较亮的像素比较暗的像素透明。 Alpha Add    Alpha添加：底层与目标层的Alpha Channles共同建立一个无痕迹的透明区域。 Luminescent Premul冷光模式：将层的透明区像素和底层作用。

AE混合模式 一、混合模式作用 使上下两层通过一定的交互计算，产生特殊效果。 二、混合模式操作 1、混合模式控制着图层与其下面的图层混合交互，除Alpha模式组以外，其他的模式运算都是通过修改源图层的颜色值产生效果。 2、 混合模式无法添加关键帧，也就是说图层不能在时间轴上用关键帧切换混合模式，如果有需要，可以在需要切换混合模式的时间点拆分图层，或者使用菜单栏-效果-通道-复合运算，效果和混合模式一样，可以实现混合模式随时间切换的功能。 三、混合模式使用 1、在图层上选择混合模式。 2、选择住图层后，按shift+“+”或“-”可以进行上下切换 四、混合模式介绍 1、基础模式组 Normal正常：当不透明度设置为100%，此合成模式将根据Alpha通道正常显示当前层，并且此层的显示不受其他层的影响。当不透明设置小于100%时，当前层的每一个象素点的衍射将受到其他层的影响，根据当前的不透明度值和其他层的色彩来确定显示的颜色。 Dissolve溶解：该合成模式将控制层与层之间的融合显示，因此该模式对于有羽化边界的层起到较大的影响。如果，当前层没有遮照羽化边界，或该层设定为完全不透明，则该模式几乎是不齐作用的。所以，该模式的最终效果将受到当前层Alpha通道的羽化程序和不透明度的影响。 Dancing Dissolve动态溶解模式：该模式同Dissolve相同。但它对融合区域进行了随机动画。 2、减少模式组 该模式组中性色为白色，按算法去掉白色后与下面的图层进行混合。 Darken变暗模式：以层颜色为准，比层颜色亮的像素被替换，而比层颜色暗的像素不改变。 Multiply正片叠底模式：将底色与层颜色相乘，形成一种光线透过两张叠在一起的幻灯片效果，结果呈现一种较暗的效果。任何颜色与黑色相乘得到黑色，与白色相乘则保持不变。 Linear Burn线性加深模式：类似于正片叠底，通过降低亮度，让底色变暗以反映混合色彩。和白色混合没有效果。 Color Burn颜色加深模式：使层的亮度减低，色彩加深。 Classic Color Burn经典颜色加深模式：为兼容早版本的Color Burn模式 3、添加模式组 中性色为黑色，按算法将黑色去掉后与下面的图层进行混合。 Add增加模式：将底色与层颜色相加，得到更为明亮的颜色。层颜色为纯黑色或底色为纯白色时，均不发生变化。 Lighten变亮模式： ​以层颜色为准，比层颜色暗的像素被替换，而比层颜色亮的像素不改变。 Screen屏幕模式： 将层的颜色互补色与底色相乘，呈现出一种较亮的效果。该模式与Multiply相反。 Linear Dodge线性减淡模式： 通过增加亮度来使得底层颜色变亮，以此获得混合色彩。与黑色混合没有任何效果。 Color Dodge颜色减淡：与Color Burn刚好相反，通过降低对比度，加亮底层颜色来反映混合色彩。与黑色混合没有任何效果。 Classic Color Dodge经典颜色减淡：兼容早版本的Color Dodge模式。 4、复杂模式组 Overlay叠加模式：该模式将根据底层的颜色，将当前层的像素进行相乘或覆盖。使用该模式可以导致当前层变亮或变暗。该模式对于中间色较为明显，对于高亮区域或者较暗区域影响不大。 Soft Light柔光模式：该模式创造一种柔和光线照射的效果，使亮度区域变得更亮，暗调区域变得更暗。如果层颜色比50%灰色暗，则图像会变暗。柔光的效果取决于层颜色。用纯黑色或者纯白色作为层颜色时，会产生明显较暗或者较亮的区域，但不会产生纯黑色或纯白色。 Hard Light强光模式：该模式创造一种强光照射的效果，使亮度区域变得更亮，暗调区域变得更暗。工作原理与Soft Light相同。 Liner Light线性加光模式：如果层颜色亮度高于50%灰色，则用增加亮度的方法来使得画面变亮，反之用降低亮度的方法来使画面变暗。 Vivid Light清晰加光模式：根据地色的加深或者减淡来增加或者减退层颜色对比度。如果底色比50%灰色亮，则层颜色降低，对比度变亮；如果底色比50%灰色暗，则层色提高，对比度变暗。 Din Light阻光模式：根据底色来替换层颜色。如果底色亮于50%灰色，则层颜色中更暗的像素被替换，亮于底色的像素不变。如果底色暗于50%灰色，则层颜色中更亮的像素被替换，暗于底色的像素不变。 Hard Mix强光混合模式：层颜色会和底色进行混合，通常的结果是亮色更加亮，暗色更加暗，降低填充不透明度建立多色调分色或者阈值，降低填充不透明度能使混合结果变得柔和。混合的颜色由底层颜色与混合图层亮度决定。 Difference差别模式：从底色中减去层颜色，或从层颜色中减去底色。这取决于哪个颜色的亮度较大（亮色减暗色）。与白色混合会使底色值反相，与黑色混合不产生变化。 Classic Difference经典差值：兼容老版本的差别模式。 Exclusion排除模式：创建一种与Difference类似但对比度较低的效果。 Hue色相模式：用底色的光度、饱和度和层颜色的色相创建结果颜色。 Saturation饱和度模式：用底色的亮度、色相和层颜色的饱和度创建结果颜色。如果底色为灰度区域，不会引起任何变化。 Color颜色模式：用底色的亮度和层颜色的饱和度、色相创建结果颜色。可以保护图像中的灰色色阶。 Luminosity高度模式：用底色的色相、饱和度和层颜色亮度的创建结果颜色。效果与颜色模式相反。该模式是除了Normal外唯一能完全消除纹理背景干扰的模式。 5、Alpha通道模式 上面所列的层模式，是通上下层的颜色通道混合产生影响变化，而下层的层模式则是通过层的Alpha通道影响混合变化。 Stencil Alpha    Alpha通道模式：该模式可以穿过Stencil层的Alpha通道显示多个层。 Stencil Luma亮度模式：该模式可以穿过Stencil层的像素显示多个层。当使用此模式时，层中较暗的像素比较亮的像素更透明。 Slihouette Alpha   Alpha通道轮廓：该模式可以通过层的Alpha通道在几层间切出一个洞。 Slihouette Luma亮度轮廓：该模式可以通过层上的像素的亮度在几层间切出一个洞，使用此模式，层中较亮的像素比较暗的像素透明。 Alpha Add    Alpha添加：底层与目标层的Alpha Channles共同建立一个无痕迹的透明区域。 Luminescent Premul冷光模式：将层的透明区像素和底层作用。
