递归例题：算24 2018-09-21|MOOC笔记C++

给出4个小于10个正整数，你可以使用加减乘除4种运算以及括 号把这4个数连接起来得到一个表达式。现在的问题是，是否存 在一种方式使得得到的表达式的结果等于24。 这里加减乘除以及括号的运算结果和运算的优先级跟我们平常 的定义一致（这里的除法定义是实数除法）。 比如，对于5，5，5，1，我们知道5 * (5 – 1 / 5) = 24，因此 可以得到24。又比如，对于1，1，4，2，我们怎么都不能得到 24。

输入
输入数据包括多行，每行给出一组测试数据，包括4个小于10个正整 数。最后一组测试数据中包括4个0，表示输入的结束，这组数据不用 处理。

输出
对于每一组测试数据，输出一行，如果可以得到24，输出“YES”； 否则，输出“NO”。

样例输入
5 5 5 1
1 1 4 2
0 0 0 0

样例输出
YES
NO

解题思路：
枚举加递归= =这个太适合枚举了。n个数算24，必有两个数要先算，无非是加减乘除，枚举。这两个数算的结果，和剩余n-2个数，就 构成了n-1个数求24的问题。边界条件即为出现一个数算24时，是就是YES，不是就是NO。

特别注意，除法会出现浮点数，浮点数比较是否相等不能使用‘==’号，而应该使用差值的绝对值是否小于某个值（很小的数字）。算浮点数绝对值的函数为fabs()

递归例题：放苹果 2018-09-21|MOOC笔记C++

把M个同样的苹果放在N个同样的盘子里，允许有的盘子空着不放， 问共有多少种不同的分法？5，1，1和1，5，1 是同一种分法。

输入
第一行是测试数据的数目t（0 <= t <= 20）。以下每行均包含二个整 数M和N，以空格分开。1<=M，N<=10。

输出
对输入的每组数据M和N，用一行输出相应的K。

样例输入
1
7 3

样例输出
8

解题思路：
设有i个苹果与k个盘子，则放法总数是f(i,k)，则k>i时，f(i,k)=f(i,i)即就算每个盘子都放一个都一定会有空盘子，则只考虑i个盘子，这时分为有空盘子的情况和没有空盘子的情况，则可以求1个空盘子的情况加上没有空盘子的情况，求1个空盘子的情况时可以拿走这一个空盘子，则变为一共有（i-1）个盘子，形成递归。

递归例题：上台阶 2018-09-21|MOOC笔记C++

用递归把问题分解为规模更小的子问题进行求解

小明爬楼梯，他可以每次走1级或者2级，输入楼梯的级数，求不同的走法数

例如：楼梯一共有3级，他可以每次都走一级，或者第一次走一级，第二次走两级，也可以第一次走两级，第二次走一级，一共三种方法。

输入，包含若干行，每行包含一个正整数N，代表楼题级数，1<=N<=30，输出不同的走法数，每一行对应一个输入

样例输入：
5
8
10

样例输出：
8
34
89

解题思路：n级台阶的走法=先走一级之后，n-1级台阶的走法+先走两级之后，n-2级台阶的走法

F(n)=F(n-1)+f(n-2)

边界条件：n<0  0; n=0 1

假设输入5，则需要求解先走一级之后4级的走法以及先走两级之后3级的走法，再进行分解。因为只有两种走法，其实也涉及了对每一走完之后下一步枚举，递归就是枚举了所有的情况，即给定一个数，每次减去1或者减去2，直到减为0，求一共有多少种减法顺序的情况。拿5来举例子，则有11111、1112、1121、121、122、2111、212、221几种情况

长得像是个递归，其实是个枚举

每次递归调用函数的时候相当于进行了一次减法，其实就是枚举

递归题目：2的幂次方表示 2018-09-12|MOOC笔记C++

总时间限制: 1000ms 

内存限制: 65536kB描述

任何一个正整数都可以用2的幂次方表示。例如：
    137=27+23+20
同时约定方次用括号来表示，即ab可表示为a(b)。由此可知，137可表示为：
    2(7)+2(3)+2(0)
进一步：7=22+2+20（21用2表示）

        3=2+20

所以最后137可表示为：
    2(2(2)+2+2(0))+2(2+2(0))+2(0)

又如：
    1315=210+28+25+2+1

所以1315最后可表示为：
    2(2(2+2(0))+2)+2(2(2+2(0)))+2(2(2)+2(0))+2+2(0)

输入一个正整数n（n≤20000）。输出一行，符合约定的n的0，2表示（在表示中不能有空格）。

样例输入
137

样例输出
2(2(2)+2+2(0))+2(2+2(0))+2(0)

递归题目：全排列 2018-09-12|MOOC笔记C++

总时间限制: 1000ms 

内存限制: 65536kB

描述
给定一个由不同的小写字母组成的字符串，输出这个字符串的所有全排列。 我们假设对于小写字母有’a’ < ‘b’ < … < ‘y’ < ‘z’，而且给定的字符串中的字母已经按照从小到大的顺序排列。输入输入只有一行，是一个由不同的小写字母组成的字符串，已知字符串的长度在1到6之间。输出输出这个字符串的所有排列方式，每行一个排列。
要求字母序比较小的排列在前面。字母序如下定义：
已知S = s1s2…sk , T = t1t2…tk，则S < T 等价于，存在p (1 <= p <= k)，使得
s1 = t1, s2 = t2, …, sp - 1 = tp - 1, sp < tp成立。

样例输入:
abc

样例输出:
abc
acb
bac
bca
cab
cba
