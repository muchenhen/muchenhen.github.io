曲线1：曲线基础 2018-12-12|cg数学方法计算机图形学

曲线在数学中的两种常用定义：

  1. 在n维空间中某区间上的连续图像
  2. 从一维空间到n维空间的连续映射

明显区别是，第一个定义中，曲线是一系列点的集合。第二个定义中，曲线是时间和点集之间的映射。

以下讨论使用第一个定义。

一条曲线是一个无限大的点集。曲线上的点具有以下性质：除了极个别的点（端点），其余点都有两个相邻点。一些曲线没有端点，可能是因为它们是无限的或者封闭的。

首先需要解决如何定义一条曲线。对于一般的曲线（非已知形状的，即非线段、圆弧等），没有经过命名的形状，称为自由曲线。自由行是的曲线很难定义，因为它们有任意的形状。

数学描述曲线的三种主要方式：

  • 隐式曲线
    通过隐式函数来定义，如

    曲线是满足上面等式的点的集合。

  • 参数曲线
    定义了从自由参数到曲线上点集的映射。即自由参数为曲线上的点提供了一个索引。曲线的参数形式是一个函数，给自由参数的每个值指派了一个位置。
    (如果认为曲线是用笔在纸上画出来的，那么，自由参数就是时间，取值范围是从开始画线到结束的时间间隔)

          参数方程是一个向量值函数，在二维情况下输出二维向量，三维情况下输出 三维向量。

  • 生成式或过程式曲线
    提供了一个过程可以生成曲线上的点。

参数形式不一定是曲线最方便的表示形式，但是参数曲线更容易画出来，因为可以在自由参数中采样。所以通常图形学中参数形式是最常用的，更容易处理。

7-31 字符串循环左移 （20 分）

输入一个字符串和一个非负整数_N_，要求将字符串循环左移_N_次。

输入格式：

输入在第1行中给出一个不超过100个字符长度的、以回车结束的非空字符串；第2行给出非负整数_N_。

输出格式：

在一行中输出循环左移_N_次后的字符串。

输入样例：

Hello World!
2

输出样例：

llo World!He

来自 <https://pintia.cn/problem-sets/14/problems/811>

#include #include using namespace std;
int main()
{
string s, t;
int N, R;
getline(cin, s);
cin >> N;
if (N > 0 && N < s.length())
{
t.append(s, 0, N);/*对临时字符串t追加s中的从下标0开始的N个字符*/
s.erase(0, N);/*删除s中的从下标0开始的N个字符*/
s.append(t);/*对s追加临时字符串t*/
}
else if (N > s.length())
{
R = N % s.length();/*取余*/
t.append(s, 0, R);
s.erase(0, R);
s.append(t);
}
cout << s;
return 0;
}

7-30 字符串的冒泡排序 （20 分）

我们已经知道了将_N_个整数按从小到大排序的冒泡排序法。本题要求将此方法用于字符串序列，并对任意给定的_K_（<_N_），输出扫描完第_K_遍后的中间结果序列。

输入格式：

输入在第1行中给出N_和_K_（1≤_K<_N_≤100），此后_N_行，每行包含一个长度不超过10的、仅由小写英文字母组成的非空字符串。

输出格式：

输出冒泡排序法扫描完第_K_遍后的中间结果序列，每行包含一个字符串。

输入样例：

6 2
best
cat
east
a
free
day

输出样例：

best
a
cat
day
east
free

来自 <https://pintia.cn/problem-sets/14/problems/810>

使用库函数可以很快完成

#include #include #include using namespace std;
int main()
{
int N, K;
cin >> N >> K;
string s[100];
for (int i = 0; i < N; i++)
cin >> s[i];
for (int i = 0; i < K; i++)
for (int j = 0; j < N - 1; j++)
if (s[j] > s[j + 1])
swap(s[j], s[j + 1]);
for (int i = 0; i < N; i++)
cout << s[i] << endl;
return 0;
}

char数组方法

这个方法是用了一些自带函数，但是gets函数在c++11中被删除，vs中保留了gets_s但是在PTA上面g++编译不能通过

#include #include using namespace std;
int main() {
char s1[81], s2[81];
gets_s(s1);
gets_s(s2);
int l1 = strlen(s1);
int l2 = strlen(s2);
char *s = strstr(s1, s2);
char word;
while (s)
{
while (s + l2 <= s1 + l1 - 1)
{
word = s[0];
s[0] = s[l2];
s[l2] = word;
s++;
}
s[0] = NULL;
s = strstr(s1, s2);
}
puts(s1);
return 0;
}
