曲线2曲线的性质 2018-12-13|cg数学方法计算机图形学

为了描述曲线，需要给出一些曲线的性质。比如描述一个圆，要给出半径和圆心的位置。但是对于自由形状的曲线，需要更加一般的性质集合来描述特定的曲线。

曲线的一些性质只归结于曲线的一个局部，其它的一些性质需要关于整个曲线的信息。

从直觉上来讲，大雾天气中的火车轨道，可以描述出它是直的或者弯曲的，是否在终点，这些就是局部性质。而对于轨道是否封闭和是否有交叉，或者轨道的长度是无法回答的，这些性质为全局性质。

局部性质是描述曲线非常重要的工具，不需要关于整个曲线的信息，局部性质包括：

  • 连续性
  • 曲线上具体地方的位置
  • 曲线上具体地方的方向
  • 曲率和其他导数

经常指定曲线要经过一个特殊的点，如果这个点是曲线的一部分，就成为曲线**插值**了一个点。如果函数f插值了一个点v，如果参数t存在一个值使f(t)=v，就把参数的值称为插值位置。

曲线1：曲线基础 2018-12-12|cg数学方法计算机图形学

曲线在数学中的两种常用定义：

  1. 在n维空间中某区间上的连续图像
  2. 从一维空间到n维空间的连续映射

明显区别是，第一个定义中，曲线是一系列点的集合。第二个定义中，曲线是时间和点集之间的映射。

以下讨论使用第一个定义。

一条曲线是一个无限大的点集。曲线上的点具有以下性质：除了极个别的点（端点），其余点都有两个相邻点。一些曲线没有端点，可能是因为它们是无限的或者封闭的。

首先需要解决如何定义一条曲线。对于一般的曲线（非已知形状的，即非线段、圆弧等），没有经过命名的形状，称为自由曲线。自由行是的曲线很难定义，因为它们有任意的形状。

数学描述曲线的三种主要方式：

  • 隐式曲线
    通过隐式函数来定义，如 曲线是满足上面等式的点的集合。

  • 参数曲线
    定义了从自由参数到曲线上点集的映射。即自由参数为曲线上的点提供了一个索引。曲线的参数形式是一个函数，给自由参数的每个值指派了一个位置。
    (如果认为曲线是用笔在纸上画出来的，那么，自由参数就是时间，取值范围是从开始画线到结束的时间间隔)

          参数方程是一个向量值函数，在二维情况下输出二维向量，三维情况下输出 三维向量。

  • 生成式或过程式曲线
    提供了一个过程可以生成曲线上的点。

参数形式不一定是曲线最方便的表示形式，但是参数曲线更容易画出来，因为可以在自由参数中采样。所以通常图形学中参数形式是最常用的，更容易处理。

插入排序 2018-10-28|C++

直接插入排序（Straight Insertion Sort）是一种最简单的排序方法，其基本操作是将一条记录插入到已排好的有序表中，从而得到一个新的、记录数量增1的有序表。

时间复杂度为O(n*n)

原理：

1、将指针指向某个元素，假设该元素左侧的元素全部有序，将该元素抽取出来，然后按照从右往左的顺序分别与其左边的元素比较，遇到比其大的元素便将元素右移，直到找到比该元素小的元素或者找到最左面发现其左侧的元素都比它大，停止；

2、此时会出现一个空位，将该元素放入到空位中，此时该元素左侧的元素都比它小，右侧的元素都比它大；

3、指针向后移动一位，重复上述过程。每操作一轮，左侧有序元素都增加一个，右侧无序元素都减少一个。

例子：
　　待比较数据：7, 6, 9, 8, 5,1

　　第一轮：指针指向第二个元素6，假设6左面的元素为有序的，将6抽离出来，形成7,,9,8,5,1，从7开始，6和7比较，发现7>6。将7右移，形成,7,9,8,5,1，6插入到7前面的空位，结果：6,7,9,8,5,1

　　第二轮：指针指向第三个元素9，此时其左面的元素6,7为有序的，将9抽离出来，形成6,7,_,8,5,1，从7开始，依次与9比较，发现9左侧的元素都比9小，于是无需移动，把9放到空位中，结果仍为：6,7,9,8,5,1

　　第三轮：指针指向第四个元素8，此时其左面的元素6,7,9为有序的，将8抽离出来，形成6,7,9,,5,1，从9开始，依次与8比较，发现8<9，将9向后移，形成6,7,,9,5,1，8插入到空位中，结果为：6,7,8,9,5,1

　　第四轮：指针指向第五个元素5，此时其左面的元素6,7,8,9为有序的，将5抽离出来，形成6,7,8,9,,1，从9开始依次与5比较，发现5比其左侧所有元素都小，5左侧元素全部向右移动，形成,6,7,8,9,1，将5放入空位，结果5,6,7,8,9,1。

第五轮：同上，1被移到最左面，最后结果：1,5,6,7,8,9。

2、相邻的扫描线与同一直线的交点位置存在递推关系，这一点可以用来把乘法换成加法

对于第一点，按照孙家广《计算机图形学》中，在处理一条扫描时，仅对与他相交的多边形的边进行求交运算。把与当前扫描线相交的边称为活性边（活动边），并把它们按与扫描线交点x坐标递增的顺序存放在一个链表中，称此链表为活性链表（AET）。例如扫描线4的“活动边表”由P1P2和P3P4两条边组成，而扫描线7的“活动边表”由P1P2、P6P1、P5P6和P4P5四条边组成。

对于第二点，由图可以轻易发现，当前的活动边已经通过直线段求交点算法求出了交点，下一条活动边的交点不需要再求交计算，通过步进关系可以直接得到新交点坐标为（x + △x, y + 1）。前面提到过，步进关系△x是个常量，与直线的斜率有关，易得出△x=-b/a，斜率的倒数。推导的过程可以参考老师上课讲的任意斜率的数值微分法。

接下来要考虑整个核心的活性边表，其实是关于扫描线的单链表。为了定义链表，首先需要定义“边”的数据类型，此处使用结构体

typedef struct tagEDGE
{
double xi;
double dx;
int ymax;
}EDGE;

需要注意的是，扫描填充在意的只是交点的x坐标。Dx用来简化求下一条活性边交点的运算。同时，如果下一条扫描线与“边”没有交点之后，需要从边表删除掉该“边”。判断是否有交点需要看扫描线y是否大于该边的两端端点的y值，所以需要定义一下边的y坐标的最大值ymax。

如果多边形不自交，关于一条“边”从当前到下一条扫描线延续，交点的顺序是不变的，有图象可以看出来。但是如果多边形自交了，比如对顶三角形，就会出现交点顺序改变，此时需要重新排序，因为扫描线是连贯的，仅有个别点需要重排序，可以使用冒泡排序：冒泡排序

这些都是书上有的东西啊！关于新交上的边肯定是要添加到表里面啊，最好插进去的时候就排好序啊，所以插入排序啊！

ScanLinePolygonFill****：在开始对每条扫描线处理之前，需要先计算出多边形的ymax和ymin并初始化“新边表”（调用下面那个）。

**FillAetScanLine()**函数执行具体的填充动作，它将AET中的边交点成对取出组成填充区间，然后根据“左闭右开”的原则对每个区间填充。

**RemoveNonActiveEdgeFromAet()**函数负责将对下一条扫描线来说已经不是“活动边”的边从AET中删除，删除的条件就是当前扫描线y与边的ymax相等，如果有多条边满足这个条件，则一并全部删除。

UpdateAndResortAet函数更新边表中每项的xi值，就是根据扫描线的连贯性用dx对其进行修正，并且根据xi从小到大的原则对更新后的AET表重新排序。

#include #include const int POINTNUM=7; //多边形点数.

//定义结构体用于活性边表AET和新边表NET
typedef struct XET
{
float x;
float dx,ymax;
XET* next;
} AET,NET;

//定义点结构体point
struct point
{
float x;
float y;
} polypoint[POINTNUM]= { {250,50},{50,150},{550,400},{250,250},{100,350},{100,100},{120,30} }; //多边形顶点

void PolyScan()
{
//计算最高点的y坐标(扫描到此结束)
int MaxY=0;
int i;
for(i=0; iMaxY)
MaxY=polypoint[i].y;

//初始化AET表
AET \*pAET=new AET;
pAET->next=NULL;

//初始化NET表
NET \*pNET\[1024\];
for(i=0; i<=MaxY; i++)
{
    pNET\[i\]=new NET;
    pNET\[i\]->next=NULL;
}
glClear(GL\_COLOR\_BUFFER\_BIT);        //赋值的窗口显示.
glColor3f(0.0,0.0,0.0);             //设置直线的颜色红色
glBegin(GL\_POINTS);
//扫描并建立NET表
for(i=0; i<=MaxY; i++)
{
    for(int j=0; jpolypoint\[j\].y)
            {
                NET \*p=new NET;
                p->x=polypoint\[j\].x;
                p->ymax=polypoint\[(j-1+POINTNUM)%POINTNUM\].y;
                p->dx=(polypoint\[(j-1+POINTNUM)%POINTNUM\].x-polypoint\[j\].x)/(polypoint\[(j-1+POINTNUM)%POINTNUM\].y-polypoint\[j\].y);
                p->next=pNET\[i\]->next;
                pNET\[i\]->next=p;
            }
            if(polypoint\[(j+1+POINTNUM)%POINTNUM\].y>polypoint\[j\].y)
            {
                NET \*p=new NET;
                p->x=polypoint\[j\].x;
                p->ymax=polypoint\[(j+1+POINTNUM)%POINTNUM\].y;
                p->dx=(polypoint\[(j+1+POINTNUM)%POINTNUM\].x-polypoint\[j\].x)/(polypoint\[(j+1+POINTNUM)%POINTNUM\].y-polypoint\[j\].y);
                p->next=pNET\[i\]->next;
                pNET\[i\]->next=p;
            }
        }
}
//建立并更新活性边表AET
for(i=0; i<=MaxY; i++)
{
    //计算新的交点x,更新AET
    NET \*p=pAET->next;
    while(p)
    {
        p->x=p->x + p->dx;
        p=p->next;
    }
    //更新后新AET先排序/
    //断表排序,不再开辟空间
    AET \*tq=pAET;
    p=pAET->next;
    tq->next=NULL;
    while(p)
    {
        while(tq->next && p->x >= tq->next->x)
            tq=tq->next;
        NET \*s=p->next;
        p->next=tq->next;
        tq->next=p;
        p=s;
        tq=pAET;
    }
    //(改进算法)先从AET表中删除ymax==i的结点
    AET \*q=pAET;
    p=q->next;
    while(p)
    {
        if(p->ymax==i)
        {
            q->next=p->next;
            delete p;
            p=q->next;
        }
        else
        {
            q=q->next;
            p=q->next;
        }
    }
    //将NET中的新点加入AET,并用插入法按X值递增排序
    p=pNET\[i\]->next;
    q=pAET;
    while(p)
    {
        while(q->next && p->x >= q->next->x)
            q=q->next;
        NET \*s=p->next;
        p->next=q->next;
        q->next=p;
        p=s;
        q=pAET;
    }
    //配对填充颜色

    p=pAET->next;
    while(p && p->next)
    {
        for(float j=p->x; j<=p->next->x; j++)
            glVertex2i(static\_cast(j),i);
        p=p->next->next;//考虑端点情况
    }
}
glEnd();
glFlush();

}
