7-15 计算圆周率 （15 分）

根据下面关系式，求圆周率的值，直到最后一项的值小于给定阈值。

​2​​π_​​=1+​3​​1​​+​3×5​​2!​​+​3×5×7​​3!​​+⋯+​3×5×7×⋯×(2_n+1)​​n!​​+⋯

输入格式：

输入在一行中给出小于1的阈值。

输出格式：

在一行中输出满足阈值条件的近似圆周率，输出到小数点后6位。

输入样例：

0.01

输出样例：

3.132157

来自 <https://pintia.cn/problem-sets/14/problems/795>

除了输出没有需要注意的

#include #include #include using namespace std;
int main() {
double a;
cin >> a;
double sum = 1.0, n = 1.0, m = 1.0, i, j;
i = m;
j = 0;
while ((n / m) >= a)
{
i = i + 2;
j = j + 1;
n = n * j;
m = m * i;
sum += n / m;
}
cout << setiosflags(ios::fixed) << setprecision(6) << 2*sum;
return 0;
}
