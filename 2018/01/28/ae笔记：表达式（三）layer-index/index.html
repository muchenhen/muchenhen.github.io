AE笔记：表达式（三）LAYER INDEX
2020-10-06|uncategorized|
Word count: 366|Reading time: 1 min

三、LAYER INDEX 1、简介：可以进行图层之间的表达式关联。 2、举例： 先建立两个完全相同的Shape Layer（形状图层），以下简称A、B A为第二层，B为第一层，对B的position属性（位置属性）添加表达式“[x,value[1]]” 制定B层的轴州属性为显示属性，X属性为变量 注意此时AE会给出error报错，因为没有给变量x赋值 然后给x进行赋值，赋值语句需要写在“[x,value[1]]” 的前一行，键入x= 注意此时依然保持输入状态，然后拖动表达式部分的拉面圈（Expression pick whip）拉至A层position属性的X值，表达式第一行就会自动补充完整变为 “x=thisComp.layer(“Shape Layer 1”).transform.position[0]” 此处需要自己加一个分号； 此时B的position属性的值会变成红色的不可修改 将表达式第一行的 “position[0]” 后面加上 “+300” ，此时表达式第一行为 “x=thisComp.layer(“Shape Layer 1”).transform.position[0]+300;” ，图层B就会相对图层A产生300的位移，修改A的X属性后B依然会和A有300的位移 继续修改表达式，将 “(“Shape Layer 1”)” 修改为 “(index+1)”，则复制B这层，复制出来的为C，C会相对与B有300的位移，继续复制C，复制出的D依然会对C有300的位移 **      注意ABCD四层必须按照序号4321，顺序为A最下B最上**
